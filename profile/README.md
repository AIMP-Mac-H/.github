## 📋 Overview

<a href="#aIMP">AIMP</a> is a free, feature-rich music player that delivers high-quality audio playback with extensive format support, powerful equalization, and deep customization — providing the audio playback capabilities that power users and audio enthusiasts want without any cost.

The <a href="#aIMP">extensive format support</a> plays over 60 audio formats natively — MP3, FLAC, WAV, AAC, OGG Vorbis, Opus, ALAC, AIFF, WMA, APE, TTA, WavPack, and many more — covering essentially any audio file format encountered in a music library without requiring format conversion or external codec installation. The <a href="#aIMP">18-band equalizer</a> provides precise frequency adjustment across the full audio spectrum with named presets for different listening scenarios and the ability to create and save custom EQ curves for different headphones, speakers, and listening preferences. Per-device EQ configuration allows different equalization settings for headphones versus desktop speakers without manual preset switching.

The <a href="#aIMP">gapless playback</a> transitions between tracks without the silence gap that interrupts album listening when each track ends — essential for classical music listeners, live album playback, and any recording where tracks flow continuously into each other without the brief silence that standard sequential playback inserts. The <a href="#aIMP">audio effects processing</a> includes tempo and pitch adjustment, reverb, stereo enhancement, speed control, and other real-time DSP effects applied to the audio stream during playback — enabling playback customization that pure player applications without DSP support cannot provide.

<a href="#aIMP">Playlist management system</a> organizes music into multiple playlists, playlist groups, and smart playlists — maintaining the organizational structure that large music libraries require for navigating and playing different content contexts. <a href="#aIMP">Cue sheet support</a> handles CUE files that describe track boundaries within large audio files — playing multi-track FLAC and WAV rips where a complete album exists as a single file with a CUE sheet defining each track's start and end position.

<a href="#aIMP">Last.fm scrobbling</a> submits played tracks to Last.fm for listening history tracking, statistics, and music discovery recommendations — maintaining the scrobble history that Last.fm users have built over years of listening. <a href="#aIMP">Highly customizable interface</a> supports skins and visual customization — changing the player's appearance through community-contributed and built-in visual themes, along with layout and interface configuration options beyond what simpler players expose.</p>
